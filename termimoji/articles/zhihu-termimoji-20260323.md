# 终端重度用户福音！这个开源工具让你在命令行里优雅地用 Emoji

作为一名程序员，你是否遇到过这些场景？

- 写 Git 提交信息时，想加个 🎉 却要在微信和终端间来回切换
- 给自己的开源项目写 README，想加个 🔥 标记亮点，搜半天找不到
- 在终端聊天或者发消息时，只能看着别人发的 ✨😎👍 干羡慕

今天推荐一个开源工具 **TermiMoji**，让你在终端里高效搜索和使用 Emoji。

## 🎯 核心功能

**闪电搜索**：内置 3000+ Emoji，支持模糊匹配，打错字也能找到

```
🔍 Search: party
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎉  party_popper    🎊  confetti_ball    🥳  partying_face    🎈  balloon
```

**一键复制**：选中即复制到剪贴板，零摩擦

**智能分类**：按关键词、情绪、活动搜索

**黑暗模式**：终端默认暗黑主题，保护眼睛

**零依赖**：纯 Python，不需要任何额外安装

## 📦 安装使用

```bash
# 安装
pip install termimoji

# 交互式搜索
termimoji

# 命令行快速搜索
termimoji "fire"     # 搜索 fire 相关
termimoji "cat" --copy  # 复制第一个结果
```

## 🤔 为什么要用这个？

作为一个经常在终端工作的人，我以前找 Emoji 的流程是：
1. 切到微信/钉钉
2. 搜索想要的 Emoji
3. 复制
4. 切回终端
5. 粘贴

这套流程打断思路不说，还很繁琐。

TermiMoji 把这个过程压缩成：输入关键词 → 选中 → 自动复制。整个过程不离开终端，丝滑流畅。
